# Руководство пользователя — Libri

## Читатель

### Вход в систему
1. Открыть приложение Libri
2. Ввести email и пароль → нажать «Войти»
3. Тестовый аккаунт: `reader@lib.ru` / `123456`

### Поиск книги
1. Открыть вкладку «Каталог»
2. Ввести название или автора в строку поиска
3. Использовать фильтры: Все / Доступные / Выданные / Забронированные

### Бронирование книги
1. Открыть карточку книги
2. Нажать «Забронировать»
3. Бронь действует 3 дня — за это время нужно забрать книгу у библиотекаря

### Просмотр выдач и штрафов
1. Открыть вкладку «Мои книги»
2. Вкладка «Выдачи» — активные книги на руках
3. Вкладка «История» — все прошлые выдачи
4. Штрафы отображаются красным — 5 руб./день просрочки

---

## Библиотекарь

### Выдача книги читателю
1. Открыть вкладку «Библиотекарь»
2. Ввести инвентарный номер экземпляра
3. Ввести ID читателя
4. Нажать «Выдать» — система проверит штрафы и лимит (макс. 5 книг)

### Возврат книги
1. Открыть вкладку «Библиотекарь»
2. Ввести инвентарный номер возвращаемого экземпляра
3. Нажать «Вернуть» — при просрочке штраф начислится автоматически

### Добавление книги в каталог
1. В каталоге нажать кнопку «+»
2. Заполнить: название, автор, ISBN, год, издательство
3. Нажать «Сохранить»

---

## Оффлайн-режим

При отсутствии интернета приложение показывает баннер «Оффлайн-режим».
Каталог и история выдач доступны из локального кэша.
При восстановлении сети данные синхронизируются автоматически.
