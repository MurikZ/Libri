# Глоссарий предметной области «Libri»

| № | Термин | Определение |
|---|---|---|
| 1 | **Book (Книга)** | Объект каталога с метаданными: название, авторы, ISBN, год издания, издательство |
| 2 | **BookInstance (Экземпляр)** | Физическая копия книги с уникальным инвентарным номером и статусом |
| 3 | **Loan (Выдача)** | Запись о временной передаче экземпляра читателю; содержит даты и статус |
| 4 | **Reservation (Бронь)** | Резервирование книги на срок до 3 дней до фактической выдачи |
| 5 | **Fine (Штраф)** | Денежное взыскание за нарушение срока возврата: 5 руб./день |
| 6 | **BookStatus** | Статус экземпляра: AVAILABLE / ON_LOAN / RESERVED / DAMAGED / LOST |
| 7 | **LoanStatus** | Статус выдачи: ACTIVE / RETURNED / OVERDUE |
| 8 | **ReservationStatus** | Статус брони: PENDING / ACTIVE / EXPIRED / CANCELLED |
| 9 | **PCMEF** | Архитектурный паттерн: Presentation–Control–Mediator–Entity–Foundation |
| 10 | **Room** | ORM-библиотека Android для работы с SQLite через аннотации Kotlin |
| 11 | **ViewModel** | Android-компонент; хранит UI-состояние при пересоздании Activity |
| 12 | **StateFlow** | Kotlin-поток данных с кэшем последнего значения для реактивного UI |
| 13 | **Repository** | Паттерн, абстрагирующий источник данных от бизнес-слоя |
| 14 | **Hilt** | Официальная библиотека внедрения зависимостей Android (Dagger 2) |
| 15 | **Jetpack Compose** | Декларативный UI-фреймворк Android от Google (Kotlin DSL) |
| 16 | **BCrypt** | Алгоритм хэширования паролей, устойчивый к перебору |
| 17 | **JWT** | JSON Web Token — стандарт аутентификации без состояния на сервере |
| 18 | **DDL** | Data Definition Language; SQL-операторы создания схемы БД |
| 19 | **3НФ** | Третья нормальная форма; исключает транзитивные зависимости в БД |
| 20 | **IDEF0** | Методология функционального моделирования; A-0 — контекстная диаграмма |
