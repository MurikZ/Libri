# UI и дизайн

## Дизайн-концепция «Тёплая библиотека»

Цветовая палитра вдохновлена кожаными переплётами и пергаментом.

## Цветовая палитра

| Название | HEX | Применение |
|---|---|---|
| Primary | `#8B7355` | Основной цвет (кнопки, акценты) |
| PrimaryVariant | `#6D5A42` | Тёмный вариант Primary |
| Secondary | `#C4A882` | Вторичный (бежевый пергамент) |
| Background | `#FAF6F0` | Фон экранов |
| Surface | `#F5EFE6` | Фон карточек |
| OnBackground | `#3D2B1F` | Основной текст |
| Accent | `#A0522D` | Акцент (сиена) |
| Error | `#C0392B` | Ошибки, просрочки |
| Success | `#6B8E6B` | Успех, доступность |

## Экраны

### LoginScreen
- Круглый логотип-аватар с эмодзи 📚
- Поля email + пароль с иконками
- Кнопка «Войти» (Primary color)
- Ссылка на регистрацию

### RegisterScreen
- TopAppBar с кнопкой «Назад»
- Поля: Имя, Фамилия, Email, Пароль, Телефон, Город
- RadioButton для выбора роли
- Валидация с сообщениями под полями

### CatalogScreen
- SearchBar с debounce 300ms
- FilterChip row: Все / Доступные / Выданные / Забронированные
- LazyColumn карточек с цветным placeholder обложки
- FAB «+» только для Библиотекаря/Администратора

### BookDetailScreen
- Полноширинный placeholder обложки (200dp) с градиентом
- Детальная информация: авторы, ISBN, год, издательство, описание
- Кнопки действий зависят от роли и статуса

### MyBooksScreen
- TabRow: Выдачи / Брони / История
- Карточки с просрочкой (красная рамка + текст)
- Секция штрафов с кнопкой «Оплатить»

### ProfileScreen
- Круговой аватар с инициалами
- Чип роли
- Список штрафов
- Кнопка «Выйти» (красная)

### LibrarianScreen
- Карточка «Выдача» с dropdown выбором читателя и книги
- Карточка «Возврат» со списком активных выдач
- Карточка «Каталог» с кнопками удаления

## Компонент BookCoverPlaceholder

Генерирует цветной градиентный placeholder на основе хэша названия книги:

```kotlin
val colors = listOf(...)
val palette = colors[title.hashCode().and(0x7FFFFFFF) % colors.size]
```

Первая буква названия отображается белым жирным шрифтом поверх градиента.
